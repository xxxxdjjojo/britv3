**quote-draft-service.ts:**
Follow the research pattern exactly. Use `@anthropic-ai/sdk` directly.

- Anthropic client singleton (check for ANTHROPIC_API_KEY, return null if not configured)
- `QuoteDraft` type: { line_items: Array<{description: string; amount: number}>, total: number, estimated_duration: string, scope_of_work: string }
- `AgentProposal` type: { valuation_range: {low: number; high: number}, comparable_properties: Array<{address: string; price: number; date: string}>, marketing_strategy: string, fee_structure: string }

- `draftTradesQuote(rfqDescription: string, rateCard: Record<string, unknown>, marketPricing: Record<string, unknown>): Promise<QuoteDraft | null>`:
  - Call Claude claude-haiku-4-5-20250315 (or claude-haiku-3-5-20241022 as fallback) with max_tokens 1024
  - System prompt: "You are a quoting assistant for a UK tradesperson. Generate competitive but fair quotes based on the trader's rate card and current market pricing. Respond with valid JSON only."
  - User prompt: template with RFQ description, rate card JSON, market pricing JSON
  - Parse JSON response, validate against QuoteDraft shape
  - 5-second timeout
  - On error: log, return null (caller shows manual form)

- `draftAgentProposal(propertyDetails: Record<string, unknown>, marketData: Record<string, unknown>): Promise<AgentProposal | null>`:
  - Similar pattern but for estate agent proposals
  - Prompt template includes property details and Land Registry comparables
  - Returns structured proposal with valuation, comparables, marketing strategy

- `getMarketPricing(serviceCategory: string, region: string)` -- query market_pricing table for the given category and region

- `getRateCard(userId: string)` -- query profiles/service_provider_details for the user's pricing JSONB

**API route POST /api/ai/quote-draft:**
- Authenticate user, verify role is 'service_provider' or 'agent'
- Body: { rfq_description, draft_type: 'trades' | 'agent', context_id? }
- For 'trades': fetch rate card + market pricing, call draftTradesQuote
- For 'agent': fetch property details + market data, call draftAgentProposal
- Return draft JSON or 200 with {draft: null, fallback: true} if AI fails
- Rate limit: max 10 AI drafts per user per day (via @upstash/ratelimit)
