   - `getTenancy(supabase, tenancyId)`: Get single tenancy.
   - `createTenancy(supabase, propertyId, data)`: Validate with tenancySchema from types/landlord.ts, insert into tenancies table. Set landlord_id from auth.uid().

   - `GET /api/portfolio` -- calls getPortfolio, returns JSON.
   - `GET /api/properties/[id]/tenancies` -- calls getTenancies with optional status query param.
   - `POST /api/properties/[id]/tenancies` -- validates body with tenancySchema, calls createTenancy, returns 201.
   - `PATCH /api/tenancies/[id]` -- validates partial body, calls updateTenancy, returns 200.
