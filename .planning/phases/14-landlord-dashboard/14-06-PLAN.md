**ComplianceUploadForm.tsx** — `"use client"` form component:
Uses react-hook-form + zod:
```typescript
const uploadCertSchema = z.object({
  property_id: z.string().uuid("Select a property"),
  category: z.enum(["gas_safety", "electrical_eicr", "epc", "insurance", "lease_agreement", "inventory"]),
  expiry_date: z.string().refine((d) => new Date(d) > new Date(), "Expiry date must be in the future"),
  document_name: z.string().min(3, "Document name required"),
  file: z.instanceof(File).refine((f) => f.size <= 5 * 1024 * 1024, "File must be under 5MB"),
});
```
Props: `{ properties: Array<{ id: string; address: string }>; defaultPropertyId?: string; defaultCategory?: string }`
(pre-filled from URL query params on the upload page)

File upload flow (private bucket — never getPublicUrl):
1. On file select: show file name + size in preview area
2. On submit:
   a. `supabase.storage.from("landlord-documents").upload(`${userId}/compliance/${propertyId}/${category}/${Date.now()}-${file.name}`, file)`
   b. If upload success: insert to `property_documents` table: `{ property_id, category, document_name, document_url: storagePath, expiry_date, landlord_id: user.id }`
   c. Set `next_reminder_date = expiry_date - 30 days` on insert
3. On success: toast "Certificate uploaded" + redirect to `/dashboard/landlord/compliance`
4. On error: toast.error with message

Use react-dropzone for file input (existing project dependency), accept: `{ "application/pdf": [".pdf"], "image/*": [".jpg", ".jpeg", ".png"] }`

Show upload progress (use react-dropzone + XMLHttpRequest progress event pattern, or Supabase storage upload progress callback).
