    provider-quote-service:
    - createQuote(supabase, providerId, input: { request_id, line_items: QuoteLineItem[], notes?, valid_until_days? }): Validates line_items (each has name, qty >= 1, unit_price >= 0), computes subtotal from line_items, inserts quotes row with status='draft', auto-generates quote number QT-{YYYY}-{NNNN} via sequence.
    - updateQuote(supabase, providerId, quoteId, updates): Only allows updates when quote status='draft'. Returns updated quote.
    - sendQuote(supabase, providerId, quoteId): Transitions quote status='sent', triggers Inngest event 'quote/sent' for email notification.
    - getQuotesByProvider(supabase, providerId, status?): Returns quotes with related service_request info, ordered by created_at DESC.

    provider-invoice-service:
    - generateInvoice(supabase, providerId, input: { booking_id?, client_id, line_items: InvoiceLineItem[], due_date_days?, notes? }): Auto-generates invoice_number INV-{YYYY}-{NNNN} using server-side sequence. Computes subtotal, VAT (default 20%), total. Inserts provider_invoices row.
    - updateInvoice(supabase, providerId, invoiceId, updates): Only when status='draft'. Recomputes totals.
    - markInvoicePaid(supabase, providerId, invoiceId, paidAt?): SET status='paid', paid_at = paidAt || now(). Returns updated invoice.
    - getInvoicesByProvider(supabase, providerId, status?): Returns ProviderInvoice[] ordered by created_at DESC.
    - Invoice and quote numbering: Use a GENERATE_INVOICE_NUMBER() / GENERATE_QUOTE_NUMBER() Supabase RPC function defined in the migration (sequences). If RPC unavailable, fall back to count-based numbering.
