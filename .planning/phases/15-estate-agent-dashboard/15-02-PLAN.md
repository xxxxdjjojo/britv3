  <action>
Create three service files following the established pattern (SupabaseClient as first param, stateless functions):

**agent-dashboard-service.ts:**
- `getAgentDashboardKpis(supabase, agentId)` — calls RPC `get_agent_dashboard_kpis` with p_agent_id, returns AgentDashboardKpis type
- `getAgentActivityFeed(supabase, agentId, limit=20)` — queries platform_events + agent_lead_activities ordered by created_at desc, returns array of activity items
- `getAgentPerformanceScore(supabase, agentId)` — queries agent_leads to compute closed/total ratio, average time-on-market for sold listings

**agent-listings-service.ts:**
- `getAgentListings(supabase, agentId, status?, page?, limit?)` — queries listings table filtered by agent_id with counts of views (from listing_analytics_events_agent), saves, and enquiries. Supports cursor-based pagination.
- `getListingAnalytics(supabase, listingId, agentId)` — queries listing_analytics_events_agent for views over time, saves, enquiries, CTR. Returns data shaped for Recharts.
- `createAgentListing(supabase, agentId, input: CreateListingInput)` — inserts into listings with agent_id set
- `updateAgentListing(supabase, listingId, agentId, input)` — updates listing, verifies agent ownership
- `archiveListing(supabase, listingId, agentId)` — soft delete (set status to 'archived')
- `restoreListing(supabase, listingId, agentId)` — restore from archived to draft

**agent-lead-service.ts:**
- `getAgentLeads(supabase, agentId, stage?)` — queries agent_leads with optional stage filter, includes related lead_activities count, ordered by updated_at desc
- `getLeadById(supabase, leadId, agentId)` — single lead with full activities timeline
- `createLead(supabase, agentId, input: CreateLeadInput)` — inserts lead, creates 'lead_created' activity
- `updateLeadStage(supabase, leadId, agentId, newStage, note?)` — updates stage, creates 'stage_changed' activity with previous/new stage
- `assignLead(supabase, leadId, agentId, assigneeId)` — updates assigned_to, creates 'lead_assigned' activity
- `getLeadActivities(supabase, leadId)` — returns all activities for a lead
- `addLeadActivity(supabase, leadId, actorId, type, description, metadata?)` — creates activity log entry

All functions throw on Supabase errors. Use type imports from @/types/agent.
