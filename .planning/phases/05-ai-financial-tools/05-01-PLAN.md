    1. Install Phase 5 dependencies from britv3.0/:
       `pnpm add @anthropic-ai/sdk @upstash/ratelimit @upstash/redis @react-pdf/renderer recharts`

    2. Create `src/services/ai/types.ts` with:
       - `AiFeature` type: `"property_description"` (union, extensible)
       - `AiCallResult` type: `{ text: string; inputTokens: number; outputTokens: number }`
       - `AiCallOptions` type: `{ feature: AiFeature; userId: string; systemPrompt: string; userMessage: string; maxTokens?: number }`
       - `AiUsageLogEntry` type for the usage tracking table shape

    3. Create `src/services/ai/claude-service.ts` with:
       - Import Anthropic from `@anthropic-ai/sdk` (reads ANTHROPIC_API_KEY from env)
       - Import Ratelimit and Redis from `@upstash/ratelimit` and `@upstash/redis`
       - Global rate limit: 100 req/min sliding window (prefix: `ai:global`)
       - Per-user rate limit: 10 req/hr sliding window (prefix: `ai:user`)
       - `getDailySpend()` function: queries ai_usage_log table for today's total cost (input_tokens * input_price + output_tokens * output_price)
       - `callClaude(options: AiCallOptions): Promise<AiCallResult | null>`:
         a. Check daily spend against AI_DAILY_SPEND_LIMIT env var (default $10)
         b. Check both rate limits (global + user)
         c. Call `client.messages.create()` with model `claude-haiku-4-5-20251001`
         d. Extract text from response content
         e. Log usage to `ai_usage_log` table via Supabase admin client
         f. Return AiCallResult or null on any failure (never throw to callers)
       - `logUsage()` helper: inserts into ai_usage_log (feature, model, input_tokens, output_tokens, user_id, created_at)
       - All errors caught and logged, returns null for graceful degradation

    4. Create `src/services/ai/claude-service.test.ts`:
       - Mock @anthropic-ai/sdk, @upstash/ratelimit, @upstash/redis, and Supabase admin client
       - Test: successful call returns text and token counts
       - Test: rate limit exceeded returns null (not throw)
       - Test: daily spend exceeded returns null
       - Test: Anthropic API error returns null
       - Test: usage is logged after successful call
       - Test: usage is NOT logged after failed call
  </action>
