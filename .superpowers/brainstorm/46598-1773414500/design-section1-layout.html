Shared Components to Build

AuthLayout (updated)
Replace current centered layout with split-panel. Left slot = children. Right slot = rotating property image + testimonial. Mobile = children only (no image panel).
OnboardingLayout
New layout for multi-step wizards. Inherits split-panel but adds step progress bar (thin brand-primary strip at top of left panel), step counter, Back + Continue footer.
OTPInput (new)
6 individual digit boxes. Auto-advance on input, paste support, backspace-to-previous, ARIA live region for screen readers. Used by 2FA pages.
PasswordStrengthMeter (upgrade)
Exists but needs visual upgrade. Animated 4-segment bar: weak/fair/good/strong. Colour transitions. Requirements checklist (8 chars, uppercase, number, symbol).
RightPanelContent
Server component. Rotating testimonial + stats. Different copy per page context (login vs signup vs onboarding). Property image from public/images/.
WizardStepper
Segmented progress bar + "Step N of M" label. Used in all 6 onboarding flows. Animated step transitions (slide-in-right).
Note on current AuthLayout: The existing layout.tsx wraps all auth pages. We'll update it in-place — same file, new split-panel structure. Pages themselves stay thin (heading + form component). Zero breaking changes.
