How users reach these pages
  • Locked: Middleware reads profiles.locked_until (set by Supabase auth rate-limiter or admin). If now < locked_until → redirect to /account-locked. Shows countdown from that timestamp.
  • Suspended: Middleware checks profiles.status === 'suspended' → redirect to /account-suspended. Shows reason from profiles.suspension_reason if set. "Contact Support" opens email. "Appeal" posts to /api/appeals/create.
  • Deletion: Initiated from Settings → Privacy → Delete Account. Sets profiles.deletion_scheduled_at = now + 30d. User gets redirect to this page on next login. "Cancel Deletion" calls /api/gdpr/cancel-deletion and clears the timestamp.
  • All three pages use the same split-panel AuthLayout — right panel shows brand content, no property testimonial (inappropriate for these states).
